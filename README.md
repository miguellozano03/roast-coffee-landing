# Roast - Coffee Landing Page

Landing Page for Roast, a fictional coffee shop chain. Explore the homepage, view our locations, browse the menu and learn about us.

## Live Demo
- Demo: https://roast-coffee-landing-kappa.vercel.app/

`The demo branch only shows mock data for portfolio purposes.`

## Technologies used

- **React** - Modern frontend and SPA
- **React Router Dom** - Routing
- **CSS** - Styling
- **Vercel** - Live deployment

## Features

- Modern landing frontend
- Sections included:
  - Home / Landing
  - Menu
  - Locations
  - About
- Responsive and mobile-first design

## Branches

- `main` -> full project with potential backend
- `demo` -> live demo with mock data

## How to run locally

1) Clone the repository:

2) Install the dependencies:
```bash
cd client/
npm install
```

3) Start the development server:
```bash
cd client/
npm run dev
```
4) Open **[http://localhost:3000](http://localhost:3000)** in your browser.
